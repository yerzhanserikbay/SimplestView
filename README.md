## Installation
### Cocoapods

To integrate Cartography into your Xcode project using CocoaPods, specify it in your Podfile:

```
target '<Your Target Name>' do
  pod 'SimplestView', '~> 1.0.0'
end
```

Then, run the following command:

```
$ pod install
```
